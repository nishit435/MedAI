# 🏥 MedAI-DPaP Multimodal LLM

MedAI-DPaP is a healthcare-focused AI application powered by multimodal Large Language Models (LLMs).  
It integrates **text, speech, and generative AI** capabilities to assist in medical tasks like answering patient queries, summarizing reports, and generating speech/audio outputs.  

The app is built using **Gradio** for an intuitive web interface and deployed on **Render** for easy access.

---

## 🚀 Features
- 🧑‍⚕️ Healthcare-focused multimodal LLM assistant  
- 🎙️ Speech-to-text support for voice queries  
- 🔊 Text-to-speech output for responses  
- 📄 Medical report analysis & summarization  
- 🌙 Clean dark-mode UI with custom styling  

---

## 🛠️ Tech Stack
- **Python 3.10+**  
- **Gradio** (frontend UI)  
- **Groq API** (LLM backend)  
- **ElevenLabs API** (text-to-speech)  
- **SpeechRecognition & PyAudio** (speech input)
